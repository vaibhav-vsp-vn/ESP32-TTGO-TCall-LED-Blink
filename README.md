# ESP32 TTGO T-Call LED Blink

This is my first GitHub project using the ESP32 TTGO T-Call board.

## Features
- LED connected to GPIO 13
- LED blinks every 10 seconds
- Simple ESP32 beginner project

## Hardware Used
- ESP32 TTGO T-Call
- LED
- Resistor

## Pin Connection

| LED | ESP32 Pin |
|-----|------------|
| +   | GPIO 13    |
| -   | GND        |
